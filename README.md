# To-Do App

A simple and user-friendly To-Do application built using Python to manage daily tasks efficiently.

##  Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Simple and clean interface

## Technologies Used

* Python

## How to Run

1. Download or clone this repository
2. Open the project folder in VS Code or terminal
3. Run the file:

```bash
python main.py
```

## Project Purpose

This project is created to practice Python programming and build a real-life useful application.
